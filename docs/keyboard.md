# NV5 键盘快捷键完整指南

NV5 是为键盘而生的笔记应用，设计灵感来自 nvALT 和 Notational Velocity。本指南列出所有可用的快捷键。

## 核心快捷键

### 笔记操作
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| ⌘N | 新建笔记 | 创建新笔记并聚焦编辑器 |
| ⌘⇧N | 用搜索词新建笔记 | 使用当前搜索词作为标题创建笔记 |
| ⌘⌫ | 删除当前笔记 | 软删除（可恢复） |
| ⌘⇧A | 切换归档状态 | 归档或取消归档当前笔记 |
| ⌘⇧T | 添加标签 | 打开标签编辑器 |

### 导出与分享
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| ⌘⇧C | 复制为 Markdown | 将笔记转为 Markdown 格式复制到剪贴板 |
| ⌘⇧R | 复制为富文本 | 将笔记转为 RTF 格式复制到剪贴板 |
| ⌘⇧T | 复制为纯文本 | 将笔记转为纯文本复制到剪贴板 |
| ⌘⇧E | 导出到文件 | 导出笔记为文件 |

### 导航与焦点
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| ⌘L | 聚焦搜索栏 | 快速搜索笔记 |
| ⌘1 | 聚焦侧栏 | 切换到侧栏（分类/标签） |
| ⌘2 | 聚焦笔记列表 | 切换到笔记列表 |
| ⌘3 | 聚焦编辑器 | 切换到编辑器 |
| ⌘B | 显示/隐藏侧栏 | 切换侧栏可见性 |
| Tab | 下一个焦点区 | 循环切换焦点区 |
| ⇧Tab | 上一个焦点区 | 反向循环切换焦点区 |

### 导航历史
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| ⌘[ | 上一个笔记 | 导航历史后退 |
| ⌘] | 下一个笔记 | 导航历史前进 |
| ⌘' | 切换到上一篇笔记 | 快速切换到之前查看的笔记 |

### 编辑器
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| ⌘⌃F | 全屏编辑器 | 隐藏侧栏和列表，专注编辑 |

### 列表导航
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| Home | 跳到列表首项 | 快速跳转到列表开头 |
| End | 跳到列表末项 | 快速跳转到列表结尾 |
| Page Up | 向上翻页 | 向上滚动约 20 项 |
| Page Down | 向下翻页 | 向下滚动约 20 项 |
| ⇧↑ / ⇧↓ | 扩展选区 | 多选笔记 |
| ⌘A | 全选 | 选中当前列表所有笔记 |

### 搜索栏
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| Esc | 清空搜索 | 第一次清空查询，第二次返回列表 |
| ↓ | 焦点到列表 | 从搜索栏直接跳到列表第一项 |
| Return | 打开第一个结果 | 切换到第一个搜索结果 |
| ⌘↑ / ⌘↓ | 历史搜索 | 在搜索历史中导航 |

### 应用
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| ⌘⇧P | 打开命令面板 | 快速访问所有命令 |
| ⌘/ | 快捷键速查表 | 显示所有快捷键的完整列表 |
| ⌘, | 偏好设置 | 打开应用设置 |

### 帮助
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| 无 | 发送反馈 | 在命令面板中搜索"反馈"或通过菜单 |
| 无 | 导出崩溃日志 | 在命令面板中搜索"崩溃日志" |

## 命令面板

按 ⌘⇧P 打开命令面板，可以：
- 搜索并执行任何命令
- 查看最近使用的命令
- 使用 ⌘↑/↓ 浏览搜索历史
- 按 Esc 清空查询或关闭面板

## 焦点管理

NV5 有四个主要焦点区：
1. **搜索栏** - 输入搜索词
2. **侧栏** - 浏览分类和标签
3. **笔记列表** - 选择笔记
4. **编辑器** - 编辑笔记内容

使用 ⌘1-3 或 Tab/⇧Tab 在焦点区之间切换。

## 标签编辑

按 ⌘⇧T 打开标签编辑器：
- 输入标签名后按 Return 添加
- 按 ⌫ 删除最后一个标签
- 点击现有标签快速添加/移除

## 全屏编辑

按 ⌘⌃F 进入全屏编辑模式，隐藏侧栏和列表。再按一次恢复。

## 提示

- 大多数快捷键可以在偏好设置中自定义
- 使用 ⌘/ 快速查看所有快捷键
- 命令面板支持模糊搜索，输入命令名称的任何部分即可找到
